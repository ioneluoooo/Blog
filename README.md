## Getting Started

### Prerequisites

1. MongoDB database
2. NodeJS working server
3. npm package manager


### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/ioneluoooo/Blog.git
   cd Back
   cd Front

2. Install dependencies

   ```bash
   npm install

3. Run the application

   ```bash
   npm start
